## Capture Guidelines

- **Resolution:** 1080 × 2400 for mobile; 1920 × 1080 for admin panel
- **Both Arabic + English** versions where applicable
- **Use realistic mock data** (no Lorem Ipsum, no test values)
- **Frame mobile screens** with device mockups for polish
- **Scrub any real personal data** (children's names, phone numbers, photos)

## Tip
Use the Figma prototype as a fallback if production builds aren't accessible — the design fidelity is high enough for showcase purposes.
