# CareConnect — One-Pager

**A Three-Tier Childcare Marketplace** *(Mother App + Babysitter App + Admin Panel)*

---

## 🎯 The Problem

Childcare in Arabic-speaking regions is dominated by informal networks (Facebook groups, word-of-mouth) with no trust verification, no reputation portability for providers, and no quality oversight. Working mothers reduce hours or exit the workforce. Skilled babysitters earn unpredictable income. Children miss enriching structured care.

## ✨ The Solution

CareConnect is a three-app ecosystem on a unified Supabase backend:
- **Mother App** — discover, book, communicate, track child wellness
- **Babysitter App** — receive bookings, execute task lists, build reputation (co-developed with @shaha123s)
- **Admin Panel** — moderate quality, manage payments, curate library

All three apps sync in real time via Supabase Realtime. Integrations: Google Maps, WhatsApp, Digital Library.

## 🧬 Built on Solid Foundations

- **IEEE 830 SRS** documentation — full requirements engineering
- **4 user personas** validated via interviews + surveys
- **RBAC + RLS** enforced at the database layer
- **GDPR & CCPA** design-aligned
- **OAuth + AES + bcrypt** for security

## 📊 Market

- **TAM:** ~$600M annual (40M households across Arab League)
- **SAM:** ~$50M (Levant + Gulf + Egypt)
- **SOM (3-year):** $1.5M–$3.5M

## 💰 Business Model

- **Family subscriptions** ($15–$69/month tiers)
- **Babysitter subscriptions** (Pro $15, Premium $29)
- **10% per-booking commission** (free-tier babysitters only)
- **Clinic/nursery white-label** partnerships
- **Future:** content licensing
